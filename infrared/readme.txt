RPiTPH Monitor基板用サンプルプログラム

【概要】
赤外線受信ユニットで受信したデータを表示し、さらに同じデータを赤外線LEDから送信するプログラムです。Raspbian Jessieでコンパイルしたバイナリとソースファイル一式です。

【ファイル構成】
infrared : バイナリ実行ファイル
main.cpp : メインルーチン
InfraredLED.cpp : 赤外線LEDを制御するクラスの実装ファイル
InfraredLED.h : 赤外線LEDを制御するクラスのヘッダーファイル
InfraredRec.cpp : 赤外線受信ユニットを制御するクラスの実装ファイル
InfraredRec.h : 赤外線受信ユニットを制御するクラスのヘッダーファイル
Makefile : g++用メイクファイル

【使い方】
お使いの赤外線機器(エアコンやテレビなど)とそのリモコンを準備します。RPiTPH Monitor基板の赤外線LEDがお使いの機器に向くように配置します。
ターミナルを開いて以下のように実行してください。
sudo ./infrared

赤外線受信待ち状態になりますので、リモコンからRPiTPH Monitor基板の赤外線受信ユニットに向けて送信してください。この時、直接リモコンからお使いの機器に赤外線が届かないように配置してください。

受信に成功すると、ターミナル上に赤外線データが表示されます。NEC及びAEHAフォーマットのみ対応しています。
さらに5秒後に、受信したデータと同じものをRPiTPH Monitor基板の赤外線LEDから送信します。お使いの機器が反応すれば正常に送信できています。赤外線リモコンのデータはお使いの機器によって異なるため、一度リモコンから入力したデータを送信するルーチンになっています。
